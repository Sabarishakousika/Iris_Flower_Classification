# Iris Flower Classification using Machine Learning

## Project Overview

This project aims to classify Iris flowers into three different species — Setosa, Versicolor, and Virginica — based on their sepal and petal measurements. A Machine Learning classification algorithm, K-Nearest Neighbors (KNN), is used to train the model and predict flower species with high accuracy.

## Objective

The objective of this project is to build a machine learning model capable of accurately identifying the species of an Iris flower using its physical characteristics.

## Dataset

The Iris dataset is one of the most widely used datasets for machine learning classification tasks.

### Dataset Information

* Total Samples: 150
* Features: 4

  * Sepal Length (cm)
  * Sepal Width (cm)
  * Petal Length (cm)
  * Petal Width (cm)
* Classes:

  * Iris Setosa
  * Iris Versicolor
  * Iris Virginica

The dataset is loaded directly from the Scikit-learn library.

## Project Workflow

1. Import required libraries.
2. Load the Iris dataset.
3. Explore and analyze the dataset.
4. Visualize feature distributions and relationships.
5. Split the dataset into training and testing sets.
6. Train the K-Nearest Neighbors (KNN) classifier.
7. Predict flower species on test data.
8. Evaluate model performance using accuracy score, confusion matrix, and classification report.

## Model Used

### K-Nearest Neighbors (KNN)

KNN is a supervised machine learning algorithm used for classification tasks. It classifies a new data point based on the majority class among its nearest neighbors.

## Results

The trained KNN model achieved high classification accuracy on the testing dataset, successfully distinguishing between the three Iris species.

Performance Metrics:

* Accuracy Score
* Precision
* Recall
* F1-Score
* Confusion Matrix
